Buyer Dashboard — Sparse-Event Variants of Option C

Buyers typically work one event per week, so 1–2 cards is the realistic case. Three layouts that don't feel empty when there's only one event to show. Each frame has a Desktop / Mobile toggle, plus a "Realistic" / "Best case" toggle that shows what the same layout looks like with 1 event vs. multiple.

1 Hero Carries the Current Event

Greeting hero swallows the in-progress (or most recent) event into a single big card. Last week appears as a slim summary strip below — no empty card if last week was a slow week.
