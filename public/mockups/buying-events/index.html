◆ Buying Events — design directions

Each mockup keeps every legacy function — they just organize it differently. Click in to see how drill-down works.

Original 5 directions

The first round. #5 was picked as the clear winner.

1
Cards · Expand-in-place
Compact event cards in a list. Click one and the full legacy panel slides open inline. Two states only: collapsed (clean) or expanded (full power).
✓ Familiar · ✓ Minimal navigation
2
Master · Detail (two-pane)
Event list on the left, full detail on the right. Mail-app style. Always one click away from any function for the selected event.
✓ Fastest switching · ✓ Always-visible context
3
List → Tabbed detail page
Click an event → full-page detail with tabs (Overview · Buyers · Travel · Marketing · Days · Cancel). Each tab is uncluttered.
✓ Most organized · ✓ Mobile-friendly
4
Kanban · Workflow columns
Events flow left to right by phase: Save the Date → Booked → Marketing → This Week → Wrapping Up. Click a card to drill in.
✓ Status at a glance · ✓ Pipeline view
5
Hub + action launchers Winner
Each event card shows KPIs (buyers, $, days). Action launchers (Travel, Marketing, etc) open in modals — keeps the page tidy.
✓ Cleanest top-level · ✓ Modal-first
6 variations on the winner (#5)

Same Hub + Launchers idea, six different visual treatments.

5a
Compact density
Same hub idea but stripped tight — thin colored bar, smaller stats, smaller chips. 3–4 events fit on screen at once.
✓ Most events visible · ✓ Power-user view
5b
Hero image
Tall hero header with logo circle and big white event name. More photographic and emotionally distinct. Each card feels like its own page.
✓ Strongest identity · ✓ Recognize by feel
5c
Side-by-side split
Identity + KPIs on the colored left panel; launcher grid on a clean white right panel. Visual story and action surface don't compete.
✓ Fast scanning · ✓ Wide-screen friendly
5d
Pill chips
Action launchers shrink to single-line rounded chips with optional count badges. Quietest variant — looks like a row of tags.
✓ Quietest · ✓ 8+ actions fit
5e
Phase timeline
5-step lifecycle bar across the top (Reserved → Booked → Marketing → In Event → Wrap up). Launchers shrink to only what's relevant in the current phase.
✓ Teaches the workflow · ✓ Phase-contextual actions
5f
Sidebar launchers
Actions move out of the body and stack as a vertical icon rail on the right edge. The body can show a "what's happening today" sentence instead.
✓ App-like · ✓ Body left for narrative
5g
Buyer avatars in header
The original #5, plus the legacy buyer-chips strip in the colored hero — overlapping initials + first names. Dashed "+" marks open slots.
✓ Who's covering this · ✓ Open slots jump out
